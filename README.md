# aviaggarwal.github.io

Personal portfolio website for Avi Aggarwal — aspiring political science and economics student.

Built as a single-page HTML/CSS/JS site with no frameworks or dependencies. Deployed via GitHub Pages.

## What's inside

- **About** — background, focus areas, and academic interests
- **Writing & Research** — selected policy briefs, op-eds, and research papers
- **Activities** — leadership roles, competitions, and extracurriculars
- **Contact** — links for outreach and professional connection

## Tech

Pure HTML5, CSS3, and vanilla JavaScript. No build tools, no npm, no frameworks. Fonts loaded from Google Fonts. Fully responsive.

## Deployment

Hosted on [GitHub Pages](https://pages.github.com/). To run locally, just open `index.html` in any browser.
